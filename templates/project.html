ForHeart: We have made a tool for the prediction of Heart Attacks using the Machine Learning Model.The website will take input parameters related to their health like blood pressure, the occurrence of chest pain, etc. from the user, and based on it, will provide the output from our model on the website.

DrLeaf: It not only makes the work easier but also reduces the effort and expenditure of the farmer to identify the disease and its treatment methods. We have to upload the image of an affected plant's leaf through our website and our plant disease prediction model predicts and returns the disease name.

GCVB: It is a software program, with which you can control Volume and Brightness levels in a Laptop or PC installed with a webcam. It can be done using simple hand gestures from a certain distance and the intensity levels can be controlled by the distance between the index finger and thumb.

MLANPR: It is a technology that uses computer vision and image processing techniques to automatically identify and recognize license plate numbers from images or video footage. This system have been developed to recognize license plates in various languages including Hindi, Tamil, Bengali, Telugu, Kannada.

Sentiment Analysis XB: It is a powerful tool for understanding how people feel about a particular topic. The approach is used to construct a pipeline that divides the incoming text into seven groups for sentiment analysis. i.e, Weakly -ve, -ve, Strongly -ve, Neutral, Weakly +ve, +ve and Strongly +ve.

Plant.iot: An IoT-based system that collects real-time data from soil and plants using Arduino, stores it in Firebase and processes it using a Python-based web app. The system provides farmers with insights into soil and plant health, helping them make informed decisions and optimize their farming practices.

MCG-51: A Morse code generator using the 8051 microcontroller is a simple yet effective communication device. The microcontroller receives the input message, processes it character by character, and then generates the corresponding Morse code patterns and transmits the encrypted data using the GSM.

Obstacle-avoiding robot: It senses and overcomes obstacles on its path. This robot has intelligence to cover the max. area of max. space. It has an ultrasonic sensor that is used to sense the obstacles coming in between the path of robot. It will move in a particular direction and avoid the obstacle which is coming in its path.

AutoMX: It is an automated machine learning tool that automates the process of selecting the best model and hyperparameters for a given dataset. AutoMX is an implementation of AutoML for quick model generation and model comparison. It can be used for both classification and regression tasks.
